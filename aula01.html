Pesquisa sobre tags HTML

Tags h1 a h6

Essas tags são utilizadas para representar titulos de seções no codigo HTML. Por padão, a tag h1 representa o maior texto, que vai diminuindo até se chega ao h6.

Tag p

A tag p é utilizada para descrever parágrafos. Esta tag fornece quebras de linha automaticamente.

Tag br

A tag br é para quebra de linha.

Tag abbr

A Tag abbr é para representar abreviações. Por exemplo:AP
