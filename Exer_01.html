Descrevendo algumas tags em HTML* de formatação de texto

Tag < em >

Esta tag é utilizada para dar ênfase ao texto contido em uma página HTML. O seu comportamento padrão exibe o texto em itálico.

Tag < strong >

Esta tag é utilizada para dar importância ao texto. O seu comportamento padrão exibe o texto em negrito.

Tags < sup > e < sub >

Esta tag é utilizada para definir texto sobrescrito e subscrito respectivamente.
Exemplo de < sup >: 23=8
Exemplo de < sub >: H2O

Tags < ins > e < del >

A tag < ins > é utilizada para definir inserções de texto. Exemplo: Dicionário A tag < del > é utilizada para remover texto. Exemplo: Dicionário

Tag < mark >

Esta tag é utilizada para marcar texto (uma especie de marca texto).
